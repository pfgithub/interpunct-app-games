usage:

create `.env` file:

```
APP_ID=
DISCORD_TOKEN=
PUBLIC_KEY=
```

run stuff

```
bun install
bun run commands # update commands
bun run start
```

start ngrok

```
ngrok http 59848
```

update app endpoint to ngrok url + `/interactions`
